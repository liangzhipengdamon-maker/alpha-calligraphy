# 四维评分体系详解

## 📊 评分标准

| 维度 | 权重 | 满分 | 评估内容 |
|------|------|------|---------|
| **笔画质量** | 25% | 25 | 线条流畅性、笔力表现、笔触质感 |
| **结构分析** | 35% | 35 | 字形比例、部件协调、布局均衡 |
| **笔画连接** | 15% | 15 | 笔画呼应、转折自然、连贯性 |
| **整体评价** | 25% | 25 | 风格统一、审美表现、书法气韵 |

## 🔍 各维度详解

### 1️⃣ 笔画质量（25%）

**定义**: 单个笔画的线条质量和笔力表现

**评估标准** (0-25分):
- **22-25分**: 笔画有力、线条流畅、笔触清晰、起笔有力、收笔果断
- **18-21分**: 笔画较有力、线条较流畅、笔触较清晰
- **14-17分**: 笔画一般、线条基本流畅、笔触不够清晰
- **10-13分**: 笔画较轻、线条不够流畅、笔触模糊
- **6-9分**: 笔画很轻、线条不流畅、笔触很模糊
- **0-5分**: 笔画无力、线条混乱、笔触不清

**关键指标**:
- 笔画粗细均匀（✅）vs 粗细不一（❌）
- 笔力表现明显（✅）vs 笔力不足（❌）
- 转折处理自然（✅）vs 生硬（❌）

**评估提示**:
```
问Gemini: "评估这个字的笔画质量，考虑：
1. 每一笔是否有力
2. 线条是否流畅
3. 笔触是否清晰
4. 起笔收笔是否有力"
```

### 2️⃣ 结构分析（35%）

**定义**: 整个字的部件组合、比例关系和空间布局

**评估标准** (0-35分):
- **31-35分**: 部件比例协调、位置准确、间距均衡、左右对称
- **26-30分**: 部件比例较协调、位置基本准确、间距基本均衡
- **21-25分**: 部件比例一般、位置基本对、间距不够均衡
- **16-20分**: 部件比例不协调、位置有偏差、间距不均匀
- **11-15分**: 部件比例差、位置明显偏差、间距很不均匀
- **0-10分**: 部件比例很差、位置错误、布局混乱

**关键指标**:
- 竖笔是否竖直（✅）vs 倾斜（❌）
- 部件是否平衡（✅）vs 头重脚轻（❌）
- 间距是否均匀（✅）vs 拥挤稀疏（❌）

**评估提示**:
```
问Gemini: "分析这个字的结构，评估：
1. 各部件的比例是否协调
2. 字的整体是否平衡
3. 间距是否均匀
4. 竖笔是否竖直
5. 左右两侧是否对称"
```

### 3️⃣ 笔画连接（15%）

**定义**: 笔与笔之间的呼应关系、转折处理、笔顺的连贯性

**评估标准** (0-15分):
- **13-15分**: 笔画呼应明显、转折自然、笔顺合理
- **10-12分**: 笔画呼应较明显、转折基本自然
- **7-9分**: 笔画呼应一般、转折不够自然
- **4-6分**: 笔画呼应不明显、转折生硬
- **1-3分**: 笔画无呼应、转折很生硬
- **0分**: 笔画完全断离

**关键指标**:
- 笔画之间是否有呼应（✅）vs 各自独立（❌）
- 转折处理是否自然（✅）vs 生硬拐角（❌）
- 笔顺是否合理（✅）vs 不合规（❌）

**评估提示**:
```
问Gemini: "评估笔画之间的连接：
1. 笔画之间是否有呼应关系
2. 转折处理是否自然流畅
3. 笔顺是否符合规范
4. 是否有明显的断离感"
```

### 4️⃣ 整体评价（25%）

**定义**: 字的整体视觉效果、风格统一度、审美表现

**评估标准** (0-25分):
- **22-25分**: 风格统一、气韵生动、审美表现好、整体和谐
- **18-21分**: 风格基本统一、气韵较生动、审美表现较好
- **14-17分**: 风格一般、气韵平平、审美表现一般
- **10-13分**: 风格不太统一、气韵较弱、审美表现一般
- **6-9分**: 风格不统一、气韵很弱、审美表现差
- **0-5分**: 风格混乱、无气韵、审美很差

**关键指标**:
- 整体视觉效果（✅ 美观）vs （❌ 难看）
- 风格是否统一（✅）vs 不统一（❌）
- 审美表现（✅ 让人舒适）vs （❌ 让人不适）

**评估提示**:
```
问Gemini: "从整体审美角度评价这个字：
1. 看起来是否舒适和谐
2. 书法风格是否统一
3. 字迹是否有气韵和灵动感
4. 整体美观度如何"
```

## 📈 总分计算

### 加权公式

```
总分 = 笔画质量分 × 0.25 + 结构分析分 × 0.35 + 笔画连接分 × 0.15 + 整体评价分 × 0.25
```

### 示例计算

**例子**: 某个字的评分

| 维度 | 得分 | 权重 | 加权分 |
|------|------|------|--------|
| 笔画质量 | 22 | 0.25 | 5.5 |
| 结构分析 | 28 | 0.35 | 9.8 |
| 笔画连接 | 12 | 0.15 | 1.8 |
| 整体评价 | 20 | 0.25 | 5.0 |
| **总分** | | | **22.1** → **22** |

## 🎯 常见评分场景

### 场景1: 笔力不足
- 笔画质量: 15分
- 结构分析: 25分
- 笔画连接: 10分
- 整体评价: 18分
- **总分: 71分（中等）**
- **建议**: 加强手指力度，多做力量练习

### 场景2: 结构不当
- 笔画质量: 22分
- 结构分析: 18分
- 笔画连接: 13分
- 整体评价: 19分
- **总分: 72分（中等）**
- **建议**: 调整部件位置和比例，学习标准笔画

### 场景3: 完美表现
- 笔画质量: 24分
- 结构分析: 33分
- 笔画连接: 14分
- 整体评价: 23分
- **总分: 94分（优秀）**
- **建议**: 保持水平，探索个人风格

## 🔧 评分调整因素

### 会增分的因素 📈
- ✅ 笔画有力、干净利落
- ✅ 结构严谨、比例协调
- ✅ 笔顺规范、转折自然
- ✅ 风格统一、气韵生动

### 会扣分的因素 📉
- ❌ 笔画无力、模糊不清
- ❌ 结构松散、比例失调
- ❌ 笔画断离、转折生硬
- ❌ 风格混乱、气韵不足
