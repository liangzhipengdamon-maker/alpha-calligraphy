# React/Vite FAQ

## ❓ 常见问题

### Q: 为什么Vite这么快？
**A**: 原生ESM支持，天然代码分割，极快的HMR

### Q: React 18有哪些新特性？
**A**: Concurrent Features、Automatic Batching、Transitions、Suspense

### Q: 如何优化React性能？
**A**: 使用memo、useMemo、useCallback，避免不必要的重渲染

### Q: TypeScript有什么优势？
**A**: 类型安全、IDE支持、重构时的安全性、代码可读性
