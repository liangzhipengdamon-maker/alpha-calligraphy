# 名师手把手 - AI硬笔书法评估系统

## 项目概述

这是一个基于AI的硬笔书法自动评估系统，利用Gemini 2.0 Flash Vision API进行智能评分，支持实时反馈和详细的评估报告。

**版本**: v0.1.0-alpha  
**状态**: 开发中（Phase 1 - AI评估功能）

## 快速开始

### 环境要求
- Node.js 18.0+
- npm 9.0+

### 安装步骤

```bash
# 1. 安装依赖
npm install

# 2. 配置环境变量
cp .env.example .env.local
# 编辑.env.local，填入你的Firebase和Gemini API密钥

# 3. 启动开发服务器
npm run dev
```

开发服务器将在 http://localhost:5173 启动

## 技术栈

- **前端框架**: React 18 + TypeScript
- **构建工具**: Vite
- **UI框架**: Tailwind CSS
- **后端服务**: Firebase (Authentication, Firestore, Storage)
- **AI模型**: Gemini 2.0 Flash Vision API
- **评分系统**: 四维评分体系（笔画质量 + 结构分析 + 笔顺规范性 + 整体评价）

## 核心功能（Phase 1）

- ✅ 硬笔书法作品上传
- ✅ AI智能评分（四维度评估）
- ✅ 评估报告生成
- ✅ 用户反馈收集
